# _TodoApp_

##### _This is a Simple  Note App with clean architecture, MVVM, Room Database, Navigation Components, Safe Args, and Search View!_

## Used Tech:


* Kotlin
* MVVM
* Room
* LiveData
* ViewModel
* ListAdapter for animations on data changes
